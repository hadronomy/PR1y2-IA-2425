Informed Search Algorithms#

The informed search algorithms are a set of algorithms that use domain-specific knowledge to find the solution. These algorithms are used when the search space is known and when the search space is small enough to explore exhaustively.

The following algorithms are implemented in this project:

  • A* Search: Explores the search space by using a heuristic function to estimate the cost of reaching the goal.

Maze.a_star([start, goal, euristic_func, ...])

Find the shortest path between the start and goal positions using the A* algorithm.

And their implementation is limited to the context of a Maze.

Maze#

The algorithms are implemented in the context of a Maze class that represents a maze.

Maze(*, rows, cols[, start, goal])

Maze data structure.

MazeTile#

Maze uses a MazeTile class to represent the tiles of the maze.

MazeTile(value[, names, module, qualname, ...])

Maze tile class.

Matrix#

Maze inherits from a Matrix class that represents a matrix and implements all the necessary operations.

Matrix(*, rows, cols, default)

Matrix data structure.

Node#

Under the hood, the algorithms use a Node class to represent the search tree.

Node([name, separator, compare_by])

Node is an extension of BaseNode, and is able to extend any Python class.
