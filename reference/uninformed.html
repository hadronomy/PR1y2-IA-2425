Uninformed Search Algorithms#

The uninformed search algorithms are a set of algorithms that do not use any domain-specific knowledge to find the solution. These algorithms are used when the search space is not known or when the search space is too large to explore exhaustively.

The following algorithms are implemented in this project:

  • Breadth-First Search (BFS): Explores the search space level by level.

UndirectedGraph.bfs(*, start, end[, ...])

Breadth-first search.

  • Depth-First Search (DFS): Explores the search space by going as deep as possible.

UndirectedGraph.dfs(*, start, end[, ...])

Depth-first search.

And their implementation is limited to the context of an UndirectedGraph.

UndirectedGraph#

The algorithms are implemented in the context of an UndirectedGraph class that represents an undirected graph.

UndirectedGraph()

Undirected graph class.

Node#

Under the hood, the algorithms use a Node class to represent the search tree.

Node([name, separator, compare_by])

Node is an extension of BaseNode, and is able to extend any Python class.
